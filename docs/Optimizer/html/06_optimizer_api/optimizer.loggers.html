Using the BasicLogger

n_best()[source]¶

Get the N best logged solutions in sorted order and their objective values.

If M < N values are logged throughout the lifetime of this object, this function will return M total values rather than N.

Return type:Tuple[ndarray, ndarray]
Returns:Tuple of arrays. The first has shape (N, d), the second (N,). Where d is the dimension of the problem being solved.
